In this section we add related information for the dongle modems LTE and LTE-A.

The following file has the "at" commands for Huawei modems to start with some basic connectivity examples:
https://support.huawei.com/enterprise/en/doc/EDOC1000097193/709abfa6/configuring-a-modem-using-at-commands

There are also GUIs in Windows to run at commands as scripts in a user friendly way.
