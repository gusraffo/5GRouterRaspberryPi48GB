Zero Tier Global Area Networking is an open source multiple VPN system for IoT with up to 50 users per VPN, that include remote access users and 
administrators as well as IoT devices.

It adds an extra layer of Security and allows administrators to use the open source Eclipse Mosquito MQTT for Topics and Subscriptions of IoT devices. Remote access can be 
via SSH for increased security or VNC that is supported via RaspberryPi. It is important to enable those remote accesses in raspbian-config


In the folling link there you can find the access to Zero Tier Global Area Networking:
https://www.zerotier.com/

In the following link you can find everything related with Eclipse Mosquito MQTT:
https://mosquitto.org/
